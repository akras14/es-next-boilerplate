# Front End Starter Projects

Collection of starter templates, all as easy to run as: 

1. Clone the branch.
2. `npm install`
3. In one terminal tab `npm run watch`
4. In another terminal tab `npm start`
5. Open your browser to `http://127.0.0.1:8080`
